# Theme Toggle (Light / Dark Mode)

A simple theme toggle application that switches between light and dark themes.  
The user's theme preference is saved using localStorage so the chosen mode persists after page reloads.  
This project demonstrates DOM manipulation, class toggling, and local storage usage.

## Features
- Switch between light and dark themes
- Modern UI with smooth transitions
- Saves user preference in localStorage
- Responsive and clean design
- Theme-aware cards and text

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## How to Run
1. Clone or download the repository.
2. Open `index.html` in any web browser.
3. Toggle the switch to change themes.

## Project Structure
- `index.html` – page layout and toggle switch
- `style.css` – theme styles and transitions
- `script.js` – toggle logic and persistence
